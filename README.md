# shrinkd

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

# Quickstart

Create an account on Supabase and create a storage bucket.
The name of the bucket can be configured with the env value `BUCKET_ID`

```
// .env
PORT=
BUCKET_ID=
SUPABASE_URL=
SUPABASE_KEY_ADMIN=
SUPABASE_KEY=
```
